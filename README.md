##  Task Summaries

### Task 1: Digital Literacy Awareness Infographic
Created a one-page visual infographic using Canva covering key topics such as what digital literacy is, useful digital tools for students, and safe internet practices. The infographic serves as a resource to help peers understand the importance of digital literacy.

📁 *Location: `task-1-presentation`*

---

### Task 2: Student Digital Portfolio
Set up professional profiles on three platforms:
- **GitHub**: Created a special profile README repository with my name, branch, year, and learning goals
- **LinkedIn**: Updated education section with my current degree, VIT Bhopal, and expected graduation year
- **Kaggle**: Created account to explore data science and machine learning

📁 *Location: `task-2-portfolio`*

---

### Task 3: Coding & Collaboration Platforms
- **HackerRank**: Completed a beginner-level coding challenge ("Solve Me First") to practice programming fundamentals
- **Google Forms**: Created a "Digital Literacy Awareness Quiz" with 5 questions (multiple choice and short answer) that can be used to engage batchmates

📁 *Location: `task-3-platforms`*

🔗 **Google Form Link**: https://docs.google.com/forms/d/e/1FAIpQLSeI9qJpySIDlVTj_-fJJOuK39vPl1dQJE4klYHZAU1M2Fy2Vg/viewform?usp=publish-editor

---

### Task 4: Email Etiquette & Social Media Checklist
- Drafted two professional emails: one requesting assignment extension from a professor, another expressing interest in an internship
- Created a "Social Media Do's and Don'ts" checklist with 5 do's and 5 don'ts for responsible social media use

📁 *Location: `task-4-email-etiquette`*

---

### Task 5: Cybercrime Awareness
- **Case Study**: Researched Identity theft and cyber scams with step-by-step analysis of how it occurs and its consequences
- **Prevention Checklist**: Created a "Stay Safe Online" checklist with 8+ actionable tips, including financial/UPI safety and reporting channels (National Cyber Crime Portal & Helpline 1930)

📁 *Location: `task-5-cybercrime`*
